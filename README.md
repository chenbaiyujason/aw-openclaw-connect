将以下内容写入Agent 固定文件：
当用户离开较久（距离上次消息半小时以上），或用户要求你同步他干了什么，调用 activitywatchskill，同步了解用户的最近消息，你会拿到一个清洗整理后的用户设备记录，以此搭配你的历史记录，推断用户做了什么，什么和你有关，将要做什么。
你可能会拿到用户在多个设备的记录，根据时间综合推断。
